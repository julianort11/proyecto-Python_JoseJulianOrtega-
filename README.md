# PROYECTO PYTHON SISTEMA CAMPUSLAND 

#### - Objetivo:

El objetivo del programa es poder manejar la información de las persona que estudian y trabajan en Campusland, llevar un control de sus horarios, notas e información, de cada uno de los de los estudiantes y trabajadores.

#### - Propósito:

El propósito principal texto en ayudar a mantener ordenada la información de todo un plantel educativo, así llevar un control de los tiempos de los estudiantes y trabajadores.

#### - Instalación 

La instalación del programa practico solo tienes que descargar la carpeta y abrirla en algún editor de código, tiene todo lo necesario para ejecutarlo después de su instalación.

#### - Funcionamiento del programa

El funcionamiento del programa esta una base de datos donde guardamos toda la información de estudiantes que están por entrar o están estudiando en campus, podemos en el programa, modificar su información, como notas, horarios, módulos (materias). También tenemos un apartado para profes donde ellos pueden ver y poner sus notas, puede ver sus horarios y clases a dar y por ultimo tenemos un menú de administradores donde estos tiene el poder de agregar el estado de un estudiante, notas, que módulos va a dictar el profesor y agregar profesores o estudiantes
